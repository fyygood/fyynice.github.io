# AI商业化落地

到底什么样的AI产品会在未来成为大家使用的主流？或者说空有强大的算力能力，但是AI其实还没有应用在我们的生活中。AI在我看来未来和当时的互联网最初的力量类似，我个人的话主要是一下几个思考的观点，后续会慢慢补充完整。

## 商业化落地场景一：AI赋能旅游

这是我在软件工程专业大二的时候就有的想法，现在非常热门的旅游app，名义上是旅游其实就是分解为定酒店定交通方式订门票的一个类似于美团的地方，真正的旅游需求没有被满足：我作为有旅行计划的一个消费者，给予我的一些限制，帮我制定一份完整详尽的旅游plan。我觉得这是大多数人想要拥有的一个功能，但是实现这个功能在AI能力问世之前有以下几个难点，或者也可以说是AI赋能旅游时要着重解决的点：

### 1. 旅游相关信息的准确性

我们经常能在网上看到很多网红打卡点，照片中和现实却大相径庭，这也是很多人在做旅游攻略的时候很头疼的问题，AI如何解决？目前想到的是一个AI负责主程序任务：生成攻略；另一个AI则负责监工，从原始数据是否准确，到另一个AI生成攻略时调用的数据是否准确。但是这样的话，可能整个程序会比较冗杂，有更轻便级别的解决方案：对于网络上的各种数据进行清洗，接入通用大模型后，通过优化prompt，或者本身产品就是一个对于用户的prompt进行加工然后让通用大模型回答的模式。

### 2. 旅游信息的爆炸式增长

随着互联网上各种旅游博主增加，旅游笔记也不断增加，数据量过多容易造成用户抉择困难，以及住宿评价数量过多，每次在预定住宿时都需要多次比较，轮番抉择，并且流动性又很强，无法实时获得消息。当AI介入后，这些数据量的问题能够快速解决。

### 3. 个性化旅游方案定制

这应该是**AI最大的卖点**！

## 具体的实现方案

1. 从省开始收集各方信息，促成最终的旅游数据图（其实这种大厂非常容易做到）。个人的话只能依靠爬虫或者其他的原始手段实现最初的数据整理。
2. 可以先把最简单的工作流程跑通，逐步实现自动化。
3. 从自动化中收集用户数据，最终产生质变。
4. 商业思考：可以只针对某些方面免费提出建议，在付费会员后，能够直接生成可以更改的攻略。
